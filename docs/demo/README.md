## How to Use These Guides

Each demo guide is self-contained and includes:

- **Prerequisites** — what must be installed or configured before starting
- **Step-by-step commands** — exact commands to run with expected output
- **Screenshot placeholders** — where to capture visuals for slide decks or recordings
- **"What to point out"** callouts — key talking points for each step
- **Acceptance criteria checklist** — proves the phase capabilities are working

### Running a Demo

1. Clone the AgenticSDLC repo and run `npm install` from the root
2. Follow the prerequisites section of the relevant guide
3. Execute each step in order — the guides are designed for a clean environment
4. Capture screenshots at each placeholder
