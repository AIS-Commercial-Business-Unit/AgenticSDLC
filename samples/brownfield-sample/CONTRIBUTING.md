# Contributing to Order Management Service

Thanks for your interest in contributing! Here's how to get involved.

## Getting Started

1. Fork the repository
2. Clone your fork locally
3. Install dependencies: `npm install`
4. Create a new branch for your changes
5. Make your changes
6. Push to your fork and open a Pull Request

## Pull Requests

- Make sure tests pass (`npm test`) before submitting
- Describe what your PR does in the description
- Link any related issues

## Reporting Bugs

Open a GitHub issue with steps to reproduce, expected behavior, and actual behavior.

## Questions

Ask in `#fulfillment-engineering` on Slack.
